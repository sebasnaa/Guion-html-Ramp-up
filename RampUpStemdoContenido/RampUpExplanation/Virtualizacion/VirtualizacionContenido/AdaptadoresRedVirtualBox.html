Adaptadores de red en VirtualBox

A cada máquina virtual creada le podemos configurar hasta 4 adaptadores de red.

Existen diferentes conexiones de red.

Adaptador puente

Las tarjetas de red de la máquina virtual se conectan automáticamente con la tarjeta de red de la máquina física. Si buscamos las máquinas están en la misma red, las máquina virtual se verían como otras máquinas físicas.

Red Interna

Haríamos una red interna propia. Ambas máquinas estarían conectadas pero no estarán conectadas a la red física real. No tendríamos acceso a internet pero las máquinas virtuales se verían entre si.

Adaptador sólo anfitrión

Nos podemos conectar desde el anfitrión a nuestras máquinas virtuales y viceversa pero no tendremos salida a internet.

NAT

Configuramos la red NAT que lleva por defecto nuestra máquina virtual. Es como si tuviésemos un pequeño router interno, que estaría conectado a nuestra tarjeta de red física. Nuestras máquinas virtuales estarían conectadas a esta red NAT.

Podríamos salir a internet pero nadie podría entrar a las máquinas. Habría que mapear las conexiones y además no tendríamos conexión entra las máquinas virtuales.

Red NAT

Si quisiéramos configurar ese router tendríamos que usar Red NAT. Nos permitiría configurar el router y la red que queremos. Tendríamos acceso a internet la diferencia es que nos permite configurar más de un “router”. Podríamos tener varias redes NAT. Sí se pueden conectar las máquinas virtuales.

Podemos crear una nueva Red NAT haciendo uso de las herramientas de VirtualBox, agregando los valores deseados.
